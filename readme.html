Traffic Challan System

Overview

The Traffic Challan System is designed to streamline the process of generating and managing traffic violation fines. It automates challan issuance, integrates data storage, and provides a user-friendly interface for enforcement authorities and vehicle owners.

Features

  • Challan Generation: Automatic generation of challans for various traffic violations.
  • Violation Database: Maintain a database of registered violations and fines.
  • Payment Integration: Online payment options for convenient fine settlement.
  • Vehicle and Driver Records: Secure storage of vehicle and driver information.
  • Search Functionality: Quick retrieval of challan or vehicle details.

Prerequisites

  • Programming Language: Python (version >= 3.8)
  • Database: MySQL or SQLite
  • Libraries:
    • Flask/Django (for web application)
    • Pandas (data manipulation)
    • NumPy (optional for calculations)
    • MySQL-connector/PySQLite (database interaction)

Installation

  1. Clone the repository: 
    git clone https://github.com/username/traffic-challan-system.git
cd traffic-challan-system
  2. Install dependencies: 
    pip install -r requirements.txt
  3. Set up the database:
    • Import the provided database.sql into your database server.
    • Update the database credentials in config.py.
  4. Run the application: 
    python app.py
  5. Access the system at http://localhost:5000.

Usage

For Authorities:

  1. Log in using your credentials.
  2. Generate a challan by entering vehicle details and selecting violations.
  3. Search and update challans as needed.

For Users:

  1. Search for challans using vehicle registration numbers.
  2. Pay fines securely through the online payment portal.

System Architecture

  • Frontend: HTML/CSS, Bootstrap, JavaScript
  • Backend: Flask/Django
  • Database: MySQL/SQLite
